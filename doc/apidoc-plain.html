Short links

Short links / create

POST /api/api/v1/shorten
Creating shortcodes

Examples

POST /api/v1/shorten
url=&shortcode=cqNArk
400
{
  "errors": "Validation failed: Url can't be blank, Shortcode has already been taken"
}

Params

  • url : required [

    Must be a String

    ]

    full url

  • shortcode : optional [

    Must be a String

    ]

    preferential shortcode

Short links / fetch_short_code

GET /api/api/v1/fetch_short_code/:shortcode
Fetches shortcode and redirects to original url

Examples

GET /api/v1/fetch_short_code/dZR1dW?dZR1dW
302
{
  "shortcode": "dZR1dW"
}

Params

  • shortcode : optional [

    Must be a String

    ]

    preferential shortcode
